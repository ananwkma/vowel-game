Add a new JS section immediately after `SECTION 5.8: ELAPSED TIMER` and before `SECTION 6: INIT`. Name it `SECTION 5.9: GIVE UP COUNTDOWN`.

```javascript
/* ============================================================
   SECTION 5.9: GIVE UP COUNTDOWN
   Per-word penalty countdown: starts at 60, counts down to 0 each second.
   When > 0: Give Up button shows "Give Up (N)".
   When 0: number disappears, Give Up is free (no penalty).
   Resets to 60 on each new word (called from initGame()).
   Penalty is read at the moment Give Up is pressed.
============================================================ */
const giveUpCountdown = (function() {
  const COUNTDOWN_START = 60;
  let current = COUNTDOWN_START;
  let intervalId = null;
  const btn = document.getElementById('give-up-btn');

  function updateButtonText() {
    if (!btn) return;
    if (current > 0) {
      btn.textContent = 'Give Up (' + current + ')';
    } else {
      btn.textContent = 'Give Up';
    }
  }

  function tick() {
    if (current > 0) {
      current -= 1;
      updateButtonText();
    } else {
      // Countdown finished — stop the interval, button stays as "Give Up"
      stop();
    }
  }

  function reset() {
    // Stop any existing countdown before starting a new one
    stop();
    current = COUNTDOWN_START;
    updateButtonText();
    intervalId = setInterval(tick, 1000);
  }

  function stop() {
    if (intervalId !== null) {
      clearInterval(intervalId);
      intervalId = null;
    }
  }

  // Returns the penalty value at the moment Give Up is pressed.
  // If countdown has reached 0, penalty is 0 (free Give Up).
  function getPenalty() {
    return current;
  }

  return { reset, stop, getPenalty };
})();
```

The updated handler:
```javascript
document.getElementById('give-up-btn').addEventListener('click', function() {
  if (gameState.phase !== 'playing') return;

  const penalty = giveUpCountdown.getPenalty(); // read before stop clears the value
  giveUpCountdown.stop();

  console.log('[GiveUp] Penalty:', penalty, '| Revealing target word:', gameState.currentWord);
  gameState.phase = 'gaveup';
  applyStateVisuals('gaveup');

  // Apply penalty to elapsed timer
  if (penalty > 0) {
    puzzleState.timerElapsed += penalty;
    // Also update the elapsedTimer's internal base so the display jumps immediately.
    // elapsedTimer.setBase re-anchors the base without restarting the interval.
    elapsedTimer.setBase(puzzleState.timerElapsed);
  }

  puzzleState.outcomes.push('gaveup');
  savePuzzleState();
  revealTargetWord();
  scheduleAutoAdvance(2000);
});
```

**Note on elapsedTimer.setBase during live timer:** `elapsedTimer.setBase()` sets `baseSeconds` and updates the display, but does not restart the interval. The timer is still running. Since `gaveup` transitions to auto-advance in 2 seconds (during which the timer continues), the base update ensures the display jumps to the correct penalized time and continues from there.

**Make elapsedTimer.setBase work while running:** In the `elapsedTimer` module from Plan 01, `setBase()` sets `baseSeconds` and updates display. Add one line to also reset `startTime` so the running offset calculation remains accurate:

If `startTime` is currently null (timer stopped), setting it here is harmless — `tick()` will only run if the interval is active, and the interval is stopped by the time `setBase` is called from the Give Up handler (since `giveUpCountdown.stop()` stops the countdown, not the elapsed timer — `elapsedTimer` keeps running until puzzle complete).
