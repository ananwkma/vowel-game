### API Contract Match

**POST /api/scores:**
- Server expects: `{ userId: string, date: string, timeMs: number }`
- Frontend sends: `{ userId: puzzleState.userId (UUID), date: puzzleState.dateKey (YYYY-MM-DD), timeMs: number (ms) }`
- ✓ Exact match

**GET /api/stats:**
- Server expects: `?date=YYYY-MM-DD&timeMs=number`
- Frontend sends: `?date=${puzzleState.dateKey}&timeMs=${timeMs}`
- ✓ Exact match

**Date format synchronization:**
- All phases use `new Date().toLocaleDateString('en-CA')` → `"YYYY-MM-DD"` ✓
