**JS — add `renderProgress()` function** in SECTION 4 (DOM Rendering), after `renderBoard()`:
```js
/**
 * Updates the Word X of 5 counter and pip dot colors from puzzleState.
 * Called by initGame() and showPuzzleComplete().
 * Pip outcomes: 'solved' → green, 'gaveup' → salmon, no outcome → neutral.
 * The current word's pip (puzzleState.wordIndex) is always neutral until completed.
 */
function renderProgress() {
  const counter = document.getElementById('word-counter');
  const pips = document.querySelectorAll('.pip');

  // wordIndex is 0-based; display as 1-based, capped at 5
  const displayIndex = Math.min(puzzleState.wordIndex + 1, 5);
  if (counter) counter.textContent = 'Word ' + displayIndex + ' of 5';

  pips.forEach((pip, i) => {
    pip.classList.remove('solved', 'gaveup');
    const outcome = puzzleState.outcomes[i];
    if (outcome === 'solved') pip.classList.add('solved');
    else if (outcome === 'gaveup') pip.classList.add('gaveup');
    // else: neutral (no class added)
  });
}
```

**Wire renderProgress() into initGame():** Add `renderProgress();` as the last statement inside `initGame()` (after `renderBoard(gameState)`).
