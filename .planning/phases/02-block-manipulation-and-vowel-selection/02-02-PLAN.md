Inside, create five child elements (e.g., `<div>` or `<span>`) for each vowel (A, E, I, O, U), each with `role="option"` and a `tabindex="-1"` (so they are not in the default tab order but can be focused programmatically).
    2.  **Add CSS Styling:** In the `<style>` block, add rules for `.vowel-picker`.
        -   Style it as a vertical bar with a distinct background.
        -   Make it `position: absolute;` so it can be placed next to a block later.
        -   Set `display: none;` by default so it is initially hidden. Add a class `.visible` that sets `display: flex;`.
        -   Style the individual vowel options, including a clear `:focus` or `.focused` style (e.g., a different background color or border) for keyboard navigation feedback.
  </action>
  <verify>
    - Inspect the DOM to confirm the `vowel-picker` element and its children exist with the correct roles.
    - Use browser dev tools to toggle the `.visible` class on the picker to check its appearance and the focus style of its options.
  </verify>
  <done>

<task type="auto">
  <name>Task 2: Implement Keyboard Navigation and Selection</name>
  <files>game.html</files>
  <action>
    In the `<script>` tag:
    1.  **Get Picker Elements:** Get references to the picker container and its vowel options.
    2.  **Create Show/Hide Functions:** Create simple `showVowelPicker()` and `hideVowelPicker()` functions. `showVowelPicker` should add the `.visible` class and programmatically focus the first vowel option. `hideVowelPicker` should remove the class.
    3.  **Implement Keyboard Listeners:** Add a `keydown` event listener to the picker container.
        -   Handle `ArrowDown` and `ArrowUp`: Prevent default behavior and move focus to the next/previous vowel option, wrapping around if necessary.
        -   Handle `Enter` and `Space`: Prevent default behavior. Identify the currently focused vowel, log its value to the console (for now), and call `hideVowelPicker()`.
        -   Handle `Escape`: Call `hideVowelPicker()`.
    4.  **Handle Clicks:** Add a `click` listener to the options to select a vowel and hide the picker.
  </action>
  <verify>

    - Press the down and up arrow keys. The focus should move between the vowels.
    - Press Enter on a vowel. The console should log the selected vowel, and the picker should disappear.
    - Press Escape. The picker should disappear.
    - Click a vowel. The console should log the selected vowel, and the picker should disappear.
  </verify>
