Cipher

How to Play

1
Each number in the quote represents a letter. Every occurrence of the same number is the same letter.
2
Tap a number block to select it (all matching numbers highlight), then type a letter to assign it.
3
You can reassign any number at any time. Assigning a letter to one number automatically frees it from any other number.
4
When every number is correctly decoded, the quote is revealed and you win!
