## Milestone Summary

**Key Decisions:**
- PNG icons generated via Node/zlib encoder — iOS apple-touch-icon requires PNG, SVG unsupported
- DailyStatus.markCompleted extended with optional extraData — cipher persists solved/percentSolved without schema changes
- Anchor count switched from char-length to word-count thresholds — more semantically meaningful difficulty scaling
- Cipher quote pool filtered to ≤12 words, ≤7-letter words — prevents overwhelming puzzles

**Issues Resolved:**
- Cipher give-up restore always showed 100% solved — markCompleted now persists solved/percentSolved
- Hunt hard-words-first path never reached results screen — checkPhaseTransition re-runs after revealCategory
- Hunt drag backtracking only 1-step — replaced with findIndex trim to any prior cell
- PWA start_url "/index.html" resolved to domain root instead of GitHub Pages subdirectory
- iOS apple-touch-icon SVG black border artifact — replaced with proper PNG

**Issues Deferred:**
- Cloud deployment for backend leaderboard (VOWEL stats local-only)
- ENH-03/ENH-04: VOWEL difficulty filter and hint system
