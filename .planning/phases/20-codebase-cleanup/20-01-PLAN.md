```json
    }

```

Note: The manifest references .png files (user will export these manually from the SVG later). The PNG files do not need to exist for the manifest to be valid — the browser will simply not show an install prompt until icons are present.

Create `icons/` directory at project root, then create `icons/lexicon-icon.svg`:

Design: A 512×512 SVG containing a 4×2 grid of letter tiles (8 tiles total spelling L-E-X-I-C-O-N with one accent tile). Each tile is a rounded rectangle (~100px wide, 110px tall) with warm palette fills. Use these colors from design-tokens.css:
- Amber primary tiles: #D4A574 fill, #1C1B18 text
- Charcoal accent tile (for visual variety): #2C2B28 fill, #F8F7F4 text
- Sage accent tile: #8BAF7C fill, #1C1B18 text
- Off-white tile: #F5F0E8 fill, #1C1B18 text

Layout: 4 columns × 2 rows, centered on 512×512 canvas with 12px gaps and 20px corner radius on each tile. Row 1: L, E, X, I (amber, charcoal, amber, amber). Row 2: C, O, N, and a small decorative diamond/star on the last tile. Letters use a bold serif or sans font (use font-family="Georgia, serif" since this is SVG — embed the letterforms directly). Font size ~52px, centered in each tile via text-anchor="middle" dominant-baseline="central".

Also create `icons/favicon.svg` — a simple single "L" letter tile (amber #D4A574 background, charcoal letter #1C1B18) at 32×32 viewBox for use as favicon.

<task type="auto">

  <files>index.html, vowel.html, ladder.html, cipher.html, hunt.html</files>
  <action>
For each of the five HTML files, update the `<head>` section to include the full PWA meta tag set. The exact tags to add/replace (some files already have partial tags — replace/merge, do not duplicate):

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no, viewport-fit=cover">
<meta name="theme-color" content="#D4A574">

<!-- Web app manifest (PWA / home screen pinning) -->
<link rel="manifest" href="/manifest.json">
<link rel="icon" type="image/svg+xml" href="/icons/favicon.svg">
<link rel="apple-touch-icon" href="/icons/lexicon-192.png">

<!-- iOS standalone mode -->
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<meta name="apple-mobile-web-app-title" content="Lexicon">

<title>[game-specific title — see below]</title>
<link rel="stylesheet" href="styles/design-tokens.css">
```
