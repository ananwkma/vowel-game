**CHANGE 3 — Add DailyStatus module to the inline `<script>`:**

Add this module definition in the `<script>` block, immediately after the closing of the `PersonalBest` object (around line 1931 of the original index.html), before the `ApiService` section:

```javascript
/* ============================================================
   SECTION: DAILY STATUS
   Writes to the shared wordGames_dailyStatus localStorage key
   so the hub can show which games have been completed today.

   localStorage namespace:
   - wordGames_dailyStatus: { vowel: { completed, dateKey, timestamp }, ... }
   - vowel_user_id, vowel_puzzle_*, vowel_best_ms, vowel_introduced: VOWEL-specific
============================================================ */
const DailyStatus = {
  KEY: 'wordGames_dailyStatus',

  markCompleted(gameId) {
    const today = new Date().toISOString().split('T')[0];
    let status = {};
    try {
      status = JSON.parse(localStorage.getItem(this.KEY) || '{}');
    } catch(e) {}
    status[gameId] = { completed: true, dateKey: today, timestamp: Date.now() };
    try {
      localStorage.setItem(this.KEY, JSON.stringify(status));
    } catch(e) {
      console.warn('[DailyStatus] Save error:', e);
    }
  }
};
```
