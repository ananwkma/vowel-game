**1. `getDateSeed()` function:**
- Read `new URLSearchParams(location.search).get('date')` — if present and matches /^\d{4}-\d{2}-\d{2}$/, use it as the seed string
- Otherwise use today's local date: `new Date().toLocaleDateString('en-CA')` which returns YYYY-MM-DD format consistently
- Store result in a module-level `const DATE_SEED` (computed once at load)
- Also compute `const IS_DEBUG = new URLSearchParams(location.search).has('debug')` (used by Plan 02)

**2. `seededRandom(seedStr)` function:**
- Simple Mulberry32-style PRNG: convert seed string to a 32-bit integer hash, then produce a generator function
- String-to-hash: `let h = 0; for (let i = 0; i < seedStr.length; i++) { h = Math.imul(31, h) + seedStr.charCodeAt(i) | 0; }` — gives a stable integer from the date string
- Generator: `let s = h >>> 0; return function() { s += 0x6D2B79F5; let t = Math.imul(s ^ s >>> 15, 1 | s); t = t + Math.imul(t ^ t >>> 7, 61 | t) ^ t; return ((t ^ t >>> 14) >>> 0) / 4294967296; }` — produces uniform [0,1) values
- Returns the generator function (call `seededRandom(DATE_SEED)` to get an rng instance)

**3. `getDailyWords()` function:**
- Filter `WordEngine.wordList` to only words that pass `WordEngine.isValidGameWord()` — this is already done by `wordList: WORDS.filter(w => w.length <= 7)` so just use `WordEngine.wordList` directly
- Create rng instance: `const rng = seededRandom(DATE_SEED)`
- Fisher-Yates shuffle of a copy of the list using rng: `const shuffled = [...WordEngine.wordList]; for (let i = shuffled.length - 1; i > 0; i--) { const j = Math.floor(rng() * (i + 1)); [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]]; }`
- Return first 5: `return shuffled.slice(0, 5)` — uniqueness guaranteed by shuffle

**4. `DailyEngine` object:**
```js
const DailyEngine = {
  dateSeed: DATE_SEED,
  isDebug: IS_DEBUG,
  dailyWords: getDailyWords(),   // computed once at load
};
```
- Log to console: `console.log('[DailyEngine] Date seed:', DATE_SEED, '| Words:', DailyEngine.dailyWords.join(', '), '| Debug:', IS_DEBUG);`
