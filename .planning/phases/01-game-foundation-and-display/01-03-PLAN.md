  <name>Visual design verification: game board display</name>
  <action>Present the completed game.html to the user for visual inspection. No automated action — this checkpoint waits for human confirmation that the Phase 1 display is correct and elegant.</action>
  <what-built>
Complete static game board display in game.html:
- HTML5 single-file structure with all CSS and JS inline
- Curated word list (1000+ words) embedded in JS
- WordEngine module (random selection, no repeats, consonant/vowel extraction)
- Elegant CSS design system: Playfair Display serif font, warm off-white background, amber vowel blocks, charcoal consonant blocks, dusty rose Give Up button
- Game board rendering: vowel placeholders (left) + consonant blocks (right) for a randomly selected word
- Give Up button: bottom-center, dusty rose styled
  </what-built>
  <how-to-verify>
1. Open `game.html` in your browser (double-click the file or drag it into Chrome/Firefox)

2. Check the overall feel:
   - Does it look elegant and sophisticated? (editorial word game, not casual mobile game)
   - Is the warm off-white background clean and airy?

3. Check the title:
   - Is "VOWEL" displayed in a serif font, uppercase, with wide letter spacing?
   - Does it feel like a wordmark, not a heading?

4. Check the blocks:
   - Are vowel placeholder blocks a warm amber/gold color with underscore "_" text?
   - Are consonant blocks a dark charcoal color with white letters?
   - Are blocks medium-sized (about 52x52px), with slightly rounded corners?
   - Do the blocks have a subtle drop shadow (soft elevation)?
   - Are the letters in a serif font that fills the tile confidently?

5. Check the layout:
   - Are vowel blocks on the LEFT of consonant blocks?
   - Are blocks horizontally centered on the page?
   - Is there comfortable spacing between blocks (8-10px gap)?

6. Check the Give Up button:
   - Is it at the bottom-center of the page?
   - Is it styled in a dusty rose/muted salmon color?
   - Does it have the same rounded corners and serif typography as the blocks?

7. Refresh the page 3-4 times:
   - Does a different word load each time?
   - Does the vowel count (amber blocks) match the word structure?

8. Check browser console (F12 → Console tab):
   - Should show: "[WordEngine] Sanity checks passed. Word list size: 1000" (or higher)
   - Should show: "[Game] New word loaded: WORDNAME | Consonants: XYZ | Vowel count: N"
   - No errors should appear in red
  </how-to-verify>
