  <action>
    Locate the `pointerup` event listener within the drag-and-drop IIFE in `game.html`.
    After `droppedElement` has been inserted into the DOM (around where `currentDraggable = null;` is set), add the `.block-settle-animation` class to `droppedElement`.
    To ensure the animation can be re-triggered, listen for the `animationend` event on `droppedElement` and remove the class once the animation completes.
    Example:
    ```javascript

    droppedElement.addEventListener('animationend', function handler() {
      droppedElement.classList.remove('block-settle-animation');
      droppedElement.removeEventListener('animationend', handler);
    });
    ```
