3. Engine initialization (runs once at page load):
   ```javascript
   const DATE_SEED = new Date().toISOString().split('T')[0] + '_cipher_v1';
   const rng = seededRandom(DATE_SEED);
   const quoteIndex = Math.floor(rng() * QUOTES.length);
   const { text: QUOTE_TEXT, author: QUOTE_AUTHOR } = QUOTES[quoteIndex];

   // Extract unique letters
   const uniqueLetters = [...new Set(
     QUOTE_TEXT.toUpperCase().replace(/[^A-Z]/g, '').split('')
   )].sort();

   // Assign numbers 1..N, then Fisher-Yates shuffle
   const numbers = uniqueLetters.map((_, i) => i + 1);
   for (let i = numbers.length - 1; i > 0; i--) {
     const j = Math.floor(rng() * (i + 1));
     [numbers[i], numbers[j]] = [numbers[j], numbers[i]];
   }

   // Build bidirectional maps
   const letterToNum = {};
   const cipherMap = {};
   uniqueLetters.forEach((letter, i) => {
     letterToNum[letter] = numbers[i];
     cipherMap[numbers[i]] = letter;
   });

   // Player state
   const guesses = {};  // number -> letter (player's current assignment)
   const gameState = { won: false };
   ```

4. buildQuoteDOM() function — tokenizes the quote and builds word-group divs:
   - Split by spaces to get words
   - For each word: strip trailing punctuation (`.`, `,`, `!`, `?`, `;`, `:`), strip leading punctuation
   - Build a `.word-group` div with `.cipher-block` elements for each letter
   - Each `.cipher-block` gets `data-number="N"` attribute (where N = letterToNum[letter.toUpperCase()]), initial class `cipher-block unsolved`, textContent = N
   - After the last block in the group, if there was trailing punctuation, append a `<span class="punct-mark">` with that punctuation character
   - Apostrophes within words (e.g., "That's") — the apostrophe is NOT encoded; render it as a `.punct-mark` span inserted between the two letter blocks at the correct position
   - Call `buildQuoteDOM(QUOTE_TEXT, letterToNum)` during initialization
